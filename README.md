# miniDash

Lodash에서 영감을 받은 경량화된 JavaScript 유틸리티 라이브러리입니다. 배열, 객체, 문자열 작업을 더 쉽고 효율적으로 만들기 위해 다양한 유틸리티 함수들을 제공합니다.

## 특징

- 배열 , 객체 , 문자열을 다루기 위한 사용하기 쉬운 API
- 외부 의존성 없는 경량화
- 누구에게나 열여있는 오픈소스

## 설치

```bash
npm i miniDash
```

## 사용법

```js
const miniDash = require('miniDash');
```
